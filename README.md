# aispring-pages

对外公开页面（隐私政策、用户协议等），通过 GitHub Pages 托管。

## 链接

- **极慢播 隐私政策**：<https://springliu9527.github.io/aispring-pages/privacy/slowplay/>
- **极慢播 支持与反馈**：<https://springliu9527.github.io/aispring-pages/support/slowplay/>（App Store 支持 URL）

## 结构

- `privacy/slowplay/` — 极慢播（Slow Play）隐私政策
- `support/slowplay/` — 极慢播 支持与反馈（用户支持、常见问题、联系方式）
- 后续可在此仓库新增其他应用或页面（如 `privacy/其他应用/`、`terms/` 等）

启用 Pages：仓库 **Settings → Pages → Source** 选 **Deploy from a branch**，Branch 选 **main**，Folder 选 **/ (root)**。
